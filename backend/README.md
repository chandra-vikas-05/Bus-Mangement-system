# Bus Booking Backend

## Environment Variables

Create a `.env` file in the root directory with:

```
PORT=5000
MONGODB_URI=mongodb://localhost:27017/bus-booking
JWT_SECRET=your_jwt_secret_key_here
NODE_ENV=development
```

## Installation

```bash
npm install
```

## Running the Server

Development mode:
```bash
npm run dev
```

Production mode:
```bash
npm start
```
